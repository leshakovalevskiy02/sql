Если на жанр ссылается книга, то при попытке удалить жанр, срабатывает ограничение
установленное при создании внешнего ключа. Может иметь значения COSCADE, SET NULL,
SET DEFAULT, RESTRICT.

Так как для удаления и обновления мной было задано ограничение SET NULL, связаная запись
при удалении той, на которую она ссылается, получила значение NULL.
